## About Apple-Sider

Apple-Sider transforms your iTunes/Apple Music Library.xml file into actionable insights with a next-generation user interface featuring:

- **Glassmorphism Design** - Modern frosted glass effects
- **Animated Gradients** - Dynamic backgrounds that respond to themes
- **Dark/Light Mode** - Persistent theme switching with keyboard shortcuts
- **Drag & Drop Upload** - Intuitive file handling with progress indicators
- **Responsive Design** - Works seamlessly across all device sizes

## Architecture

Built with:
- **Backend**: Flask (Python)
- **Frontend**: Modern JavaScript with CSS3 animations
- **Containerization**: Docker with Docker Compose
- **Documentation**: Fumadocs framework
